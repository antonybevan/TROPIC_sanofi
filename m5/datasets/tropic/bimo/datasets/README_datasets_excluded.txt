DATASETS EXCLUDED FROM THIS DATA-FREE PREVIEW
=============================================

This eCTD Module 5 tree was assembled in PREVIEW (data-free) mode:
    python3 06_telemetry/package_ectd.py --preview

The SAS Transport (XPORT v5, *.xpt) datasets that belong in this folder are
deliberately NOT included. They are de-identified, patient-level clinical-trial
data obtained via Project Data Sphere under a Data Use Agreement that does not
permit public redistribution; the project therefore never commits row-level data.
The co-located define.xml fully describes the datasets that would be present here
(structure, variables, controlled terminology, and derivations/origins).

To materialise the datasets locally (with the licensed source data present):
    python3 06_telemetry/package_ectd.py
